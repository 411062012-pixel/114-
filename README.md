# 114年高雄市不動產拍賣統計網站

這是一個展示民國114年（2025年）1月至10月高雄市各行政區不動產拍賣統計的專業一頁式網站。

## 功能特色

- 📊 **統計概覽**：總土地筆數、總建物棟數統計
- 📈 **互動圖表**：橫向堆疊條形圖展示各行政區數據
- 📋 **詳細表格**：完整數據表格，按總計排序
- 📱 **響應式設計**：支援各種裝置瀏覽
- 🇹🇼 **繁體中文**：完整繁體中文介面

## 資料範圍

- **統計期間**：民國114年1月至10月
- **行政區數量**：38個行政區
- **資料類型**：土地筆數、建物棟數

## 使用方式

直接在瀏覽器中開啟 `kaohsiung-auction-stats.html` 即可查看。

## 技術架構

- HTML5
- CSS3（漸層背景、動畫效果）
- JavaScript（Chart.js 圖表庫）

## GitHub Pages 部署

此網站可直接透過 GitHub Pages 部署：

1. 將此倉庫推送到 GitHub
2. 在倉庫設定中啟用 GitHub Pages
3. 選擇 main 分支作為來源
4. 網站將自動部署到 `https://[你的用戶名].github.io/[倉庫名稱]/`
